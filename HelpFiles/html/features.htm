New features in this release

MB Software® Paint 2000 version 1.0 contains these new features:

  • Create, save and open bmp files.
  • Draw lines, oval, pies, arcs, text and a lot more.
  • Learn how to use a simple paint application.
  • Learnable Source codes available thru registering the software.

NOTE: These enhancements are designed to make Paint 2000 fully user friendly.
