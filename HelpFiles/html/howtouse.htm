How to Use the program.

   To draw a line simply click on the Line Image at the Tools Toolbar and press your left mouse button down and drag it to how long you want the line to be and release the left mouse button. This all applies to drawing rectangles, ellipse, oval, arc, pie, round rectangle, eraser, and freehand. To create a text simply click on the A image button on the Tools toolbar. You will be prompt for a Font Dialog box.

 After this, you should click on the Client Area where you want the text to be displayed. You will then again be asked to type what text you want to be displayed on another Dialog box.

 After you click ok, your text will be displayed. To fill drawings simply click on the Fill Image on the Tools Toolbar. You will then be prompt for a Colour Dialog box, which you can either select what custom brush you want or use the default selected brush. See Image next page.

  • To change brush colours, simply press one button in the Brush Background Toolbar.
  • To change pen colours, simpley press one button in the Pen Color Toolbar located at the ReBar.
  • To change pen sizes, simply press one button in the Pen Size Toolbar located at the ReBar.
  • You can also enter custom sizes by clicking on the Custom size image on the Pen Size Toolbar. You will be prompt for a Dialog box that has a slider control on it that you can drag and change.  You can then override the custom size by typing a value lower to the slider control at the Edit box.
  • You can show and hide toolbars at the Option menu.
  • You can choose your termination button at the Option menu too.
  • You can enable Undo/Redo on the Option menu.
  • You can fix the Flickering on the Option menu.
  • You can show the splash screen on the Option menu.
  • You can zoom in, zoom out and reset using the View menu.
  • You can use undo / redo using the Edit menu.
  • You can Save, Create New, Open, Setup Printer, and Exit using the File menu.
  • Also this entire menu item has a default menu trigger and accellerators.
  • The menu items dynamically checks and unchecks whenever you check an item.  The toolbar items dynamically press and unpress whenever you press a button.
