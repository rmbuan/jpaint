Introduction

            Welcome to Paint 2000 v1.0, a Paint like program using VC++/MFC.

            This program is made with Visual C++ using MFC. The purpose of this program is to introduce students to programming in Visual C++ using MFC, using the same basic programming techniques that Microsoft used on their Paintbrush program. That is the GDI and using SDI interfaces. As far as the program is concerned this is still in major development.

            Remember to visit the Paint 2000 homepage at http://mbsoft.dhs.org/ or one of its mirror sites regularly for the latest information, as well as for links to other Paint 200 and Graphics related information and websites.

            Good luck and have fun!
